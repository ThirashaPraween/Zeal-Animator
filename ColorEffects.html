Zeal Animator Color Effects

Color Animation Effects and use background and text both elements with specific class. When you use this color effects, you have to use other optional classes for animation timing and animation repetition.

# Standard


RGB
CMYK

Standard Effects

    # Here zt-5 class for animation time 5s, and z-repeat class for repeat the same animation. 
    # Also, the element should have valid sizes.             

    <div class="z-bg-rgb zt-5 z-repeat">RGB<div/>
    <div class="z-bg-cmyk zt-5 z-repeat">CMYK<div/>
