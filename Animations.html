Zeal Animator Animations

Animations can use with any html element. When you use this animations, you have to use other optional classes.

# Pulse


Pulse
O Pulse
Pulse Hover

Pulse

  # Here zt-2 class for animation time 2s, and z-repeat class for repeat the same animation.            

  <div class="z-pulse zt-2 z-repeat">Pulse<div/>
  <div class="z-opulse zt-2 z-repeat">O Pulse<div/>

  # ztt-1 class for animation transition time 1s.    

  <div class="zhover-pulse ztt-1">Pulse Hover<div/>
