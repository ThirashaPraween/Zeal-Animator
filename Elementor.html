# Loading Bubbles



Buttons

        # To apply bubbles. use z-loader-el class.
        # If you want more bubbles, try to add more z-loader-el class div tags.
    
        <div class="z-loader">
          <div class="z-loader-el"></div>
          <div class="z-loader-el"></div>
        </div>

# Buttons



Buttons

        <button class="z-btn z-ripple-back">Back Button</button>
        <button class="z-btn z-ripple-for">Forward Button</button>
        <button class="z-btn z-ripple-both">Ultra Button</button>
        <button class="z-btn z-ripple-down z-shadow">Down 3D Button</button>
